9 qubits code (Shor's code)

9 qubits code (Shor's code)

Mar 15, 20254 min read

Using encoding on 9 Qubits, it is possible to correct any kind of noise induced on a single qubit.
The 9 qubits code encompasses both the 3 qubits code and incidentally this leads to general error correction on a single qubit.
The qubit we want to encode is:

∣ψ⟩=α∣0⟩+β∣1⟩\ket{\psi} = \alpha \ket{0} +\beta \ket{1}∣ψ⟩=α∣0⟩+β∣1⟩

Label the 9 physical qubits as 0,1,…,8. The basis for the logical qubit is:

∣0⟩L≡(∣000⟩+∣111⟩2)⊗3\ket{0}_L \equiv\left(\frac{\ket{000}+\ket{111}}{\sqrt{2}}\right)^{\otimes 3}∣0⟩L​≡(2​∣000⟩+∣111⟩​)⊗3 ∣1⟩L≡(∣000⟩−∣111⟩2)⊗3\ket{1}_L \equiv\left(\frac{\ket{000}-\ket{111}}{\sqrt{2}}\right)^{\otimes 3}∣1⟩L​≡(2​∣000⟩−∣111⟩​)⊗3 ∣ψ⟩L=α∣0⟩L+β∣1⟩L\ket{\psi }_L = \alpha \ket{0}_L + \beta \ket{1}_L∣ψ⟩L​=α∣0⟩L​+β∣1⟩L​

Encoding

We use a sequence of Hadamard Gates and CNOT gates. Basically, the first step is the encoding as seen in the phase-flip error correction on 3 qubits, while the second step is the encoding as seen in the bit-flip for each of these 3 qubits (so 9 qubits total).
The logical qubit is:

∣ψ⟩L=α(∣000⟩+∣111⟩2)⊗3+β(∣000⟩−∣111⟩2)⊗3\ket{\psi}_L = \alpha\left(\frac{\ket{000}+\ket{111}}{\sqrt{2}}\right)^{\otimes 3}+\beta\left(\frac{\ket{000}-\ket{111}}{\sqrt{2}}\right)^{\otimes 3}∣ψ⟩L​=α(2​∣000⟩+∣111⟩​)⊗3+β(2​∣000⟩−∣111⟩​)⊗3

Error syndrome measurement

The measurements we need to perform are:

  1. Sequence of Z measurements:
    • Z0⊗Z1Z_0\otimes Z_1Z0​⊗Z1​
    • Z1⊗Z2Z_1\otimes Z_2Z1​⊗Z2​
    • Z3⊗Z4Z_3\otimes Z_4Z3​⊗Z4​
    • Z4⊗Z5Z_4\otimes Z_5Z4​⊗Z5​
    • Z6⊗Z7Z_6\otimes Z_7Z6​⊗Z7​
    • Z7⊗Z8Z_7\otimes Z_8Z7​⊗Z8​
  2. Sequence of X measurements:
    • Ω0≡X0X1X2X3X4X5\Omega _0 \equiv X_0X_1X_2X_3X_4X_5Ω0​≡X0​X1​X2​X3​X4​X5​
    • Ω1≡X3X4X5X6X7X8\Omega _1 \equiv X_3X_4X_5X_6X_7X_8Ω1​≡X3​X4​X5​X6​X7​X8​
      The error correction protocol is the following:
  3. The Z measurements commute and are used to detect bit-flip errors for each triplet (see Bit-flip error - 3 qubits)
  4. Correct bit-flips if found
  5. The Ωi\Omega _iΩi​ commute. These two measurements provide two bits of information: whether a single phase-flip occurred and in which triplet (we don’t need more bits of info as phase flip e.g. in 0,1,2 are mutually indistinguishable in our code)
  6. Correct phase-flip if found

Noise at Stinespring dilation level

At the Stinespring dilation level, any qubit channel is almost a Pauli channel

N(ρ)=trE(USE(ρS∣0⟩E⟨0∣)USE†)\mathcal{N} (\rho) = \tr _E \left(U_{SE}(\rho _S \ket{0}_E\bra{0})U_{SE} ^\dagger\right)N(ρ)=trE​(USE​(ρS​∣0⟩E​⟨0∣)USE†​)

Before tracing out the environment, for Pure states we have

∣ψ⟩S↦USE∣ψ⟩S∣0⟩E\ket{\psi}_S \mapsto U_{SE} \ket{\psi}_S\ket{0}_E∣ψ⟩S​↦USE​∣ψ⟩S​∣0⟩E​ ∣ψ⟩=α∣0⟩+β∣1⟩\ket{\psi} = \alpha \ket{0} + \beta \ket{1}∣ψ⟩=α∣0⟩+β∣1⟩

We define the (not normalized) vectors:

∣gij⟩E≡S⟨i∣USE∣j⟩S∣0⟩E\ket{g _{ij}}_E \equiv {}_S\bra{i}U_{SE}\ket{j}_S\ket{0}_E∣gij​⟩E​≡S​⟨i∣USE​∣j⟩S​∣0⟩E​

so that:

∣ψ⟩S↦α∣0⟩S∣g00⟩E+α∣1⟩S∣g10⟩E+β∣0⟩S∣g01⟩E+β∣1⟩S∣g11⟩E\ket{\psi } _S \mapsto \alpha\ket{0}_S\ket{g_{00}}_E+\alpha\ket{1}_S\ket{g_{10}}_E+\beta\ket{0}_S\ket{g_{01}}_E+\beta\ket{1}_S\ket{g_{11}}_E∣ψ⟩S​↦α∣0⟩S​∣g00​⟩E​+α∣1⟩S​∣g10​⟩E​+β∣0⟩S​∣g01​⟩E​+β∣1⟩S​∣g11​⟩E​

This can be rewritten as

USE∣ψ⟩S∣0⟩E=∣g00⟩E+∣g11⟩E2∣ψ⟩S+∣g00⟩E−∣g11⟩E2Z∣ψ⟩S+∣g10⟩E+∣g01⟩E2X∣ψ⟩S+∣g10⟩E−∣g11⟩E2iY∣ψ⟩SU_{SE} \ket{\psi}_S\ket{0}_E = \frac{\ket{g_{00}}_E+\ket{g_{11}}_E}{2}\ket{\psi}_S + \frac{\ket{g_{00}}_E-\ket{g_{11}}_E}{2}Z\ket{\psi}_S+ \frac{\ket{g_{10}}_E+\ket{g_{01}}_E}{2}X\ket{\psi}_S+ \frac{\ket{g_{10}}_E-\ket{g_{11}}_E}{2}iY\ket{\psi}_SUSE​∣ψ⟩S​∣0⟩E​=2∣g00​⟩E​+∣g11​⟩E​​∣ψ⟩S​+2∣g00​⟩E​−∣g11​⟩E​​Z∣ψ⟩S​+2∣g10​⟩E​+∣g01​⟩E​​X∣ψ⟩S​+2∣g10​⟩E​−∣g11​⟩E​​iY∣ψ⟩S​

We can label the vectors of the environment for easier reading:

USE∣ψ⟩S∣0⟩E=∣1⟩E⊗∣ψ⟩S+∣Z⟩E⊗Z∣ψ⟩S+∣X⟩E⊗X∣ψ⟩S+∣Y⟩E⊗Y∣ψ⟩SU_{SE} \ket{\psi}_S\ket{0}_E = \ket{\mathbb{1}}_E\otimes\ket{\psi}_S + \ket{Z}_E\otimes Z\ket{\psi}_S+ \ket{X}_E\otimes X\ket{\psi}_S+ \ket{Y}_E \otimes Y\ket{\psi}_SUSE​∣ψ⟩S​∣0⟩E​=∣1⟩E​⊗∣ψ⟩S​+∣Z⟩E​⊗Z∣ψ⟩S​+∣X⟩E​⊗X∣ψ⟩S​+∣Y⟩E​⊗Y∣ψ⟩S​ USE∣ψ⟩S∣0⟩E=∣1⟩E⊗∣ψ⟩S+∣Z⟩E⊗Z∣ψ⟩S+∣X⟩E⊗X∣ψ⟩S−∣Y⟩E⊗XZ∣ψ⟩SU_{SE} \ket{\psi}_S\ket{0}_E = \ket{\mathbb{1}}_E\otimes\ket{\psi}_S + \ket{Z}_E\otimes Z\ket{\psi}_S+ \ket{X}_E\otimes X\ket{\psi}_S- \ket{Y}_E \otimes XZ\ket{\psi}_SUSE​∣ψ⟩S​∣0⟩E​=∣1⟩E​⊗∣ψ⟩S​+∣Z⟩E​⊗Z∣ψ⟩S​+∣X⟩E​⊗X∣ψ⟩S​−∣Y⟩E​⊗XZ∣ψ⟩S​

When tracing the environment, we only obtain a Pauli channel if the vectors of the environment are orthogonal, otherwise mixed terms are found.

Error correction

We use the modified version of the Generic Error Correction module:

D≡∑k∣k⟩A⟨k∣⊗Dk\mathcal{D} \equiv \sum _k \ket{k}_A\bra{k}\otimes\mathcal{D}_kD≡k∑​∣k⟩A​⟨k∣⊗Dk​

Realistically (in the lab), it’s easier to do classical control, quantum control is easier to deal with mathematically. Also, performing a measurement on A after the quantum controlled gate D\mathcal{D}D is the same as performing the standard QEC.
We now apply this formalism to the 9 qubit code

Let’s see what happens at each stage. Stages 2-3 correct bit flip error and 4-5 correct phase flip error.

∣Ψ⟩0=∣ψ⟩L∣0⟩E∣0⟩A1∣0⟩A2\ket{\Psi} _0 = \ket{\psi}_L\ket{0}_{E}\ket{0}_{A_1}\ket{0}_{A_2}∣Ψ⟩0​=∣ψ⟩L​∣0⟩E​∣0⟩A1​​∣0⟩A2​​ ∣Ψ⟩1=(USE∣ψ⟩L∣0⟩E)∣0⟩A1∣0⟩A2\ket{\Psi} _1 = \left(U_{SE} \ket{\psi} _L \ket{0} _E\right) \ket{0}_{A_1}\ket{0}_{A_2}∣Ψ⟩1​=(USE​∣ψ⟩L​∣0⟩E​)∣0⟩A1​​∣0⟩A2​​

At stage 2, each time there is a bit flip (X), that information is registered in A1A_1A1​

∣Ψ⟩2 =∣ψ⟩L∣1⟩E∣0⟩A1∣0⟩A2+XS∣ψ⟩L∣X⟩E∣XS⟩A1∣0⟩A2−XSZS∣ψ⟩L∣Y⟩E∣XS⟩A1∣0⟩A2+ZS∣ψ⟩L∣Z⟩E∣0⟩A1∣0⟩A2\begin{split} \ket{\Psi}_2 \,=\quad &\ket{\psi}_L\ket{\mathbb{1}}_{E}\ket{0}_{A_1}\ket{0}_{A_2} + X_S\ket{\psi}_L\ket{X}_{E}\ket{\color{violet}X_S}_{A_1}\ket{0}_{A_2}-\\ & X_S Z_S\ket{\psi}_L\ket{Y}_{E}\ket{\color{violet}X_S}_{A_1}\ket{0}_{A_2} + Z_S\ket{\psi}_L\ket{Z}_{E}\ket{0}_{A_1}\ket{0}_{A_2} \end{split}∣Ψ⟩2​=​∣ψ⟩L​∣1⟩E​∣0⟩A1​​∣0⟩A2​​+XS​∣ψ⟩L​∣X⟩E​∣XS​⟩A1​​∣0⟩A2​​−XS​ZS​∣ψ⟩L​∣Y⟩E​∣XS​⟩A1​​∣0⟩A2​​+ZS​∣ψ⟩L​∣Z⟩E​∣0⟩A1​​∣0⟩A2​​​

At stage 3, if A1A_1A1​ is in the ∣XS⟩\ket{X_S}∣XS​⟩ state, apply XSX_SXS​ (meaning on the right position in the triplet).

∣Ψ⟩3 =∣ψ⟩L∣1⟩E∣0⟩A1∣0⟩A2+∣ψ⟩L∣X⟩E∣XS⟩A1∣0⟩A2−ZS∣ψ⟩L∣Y⟩E∣XS⟩A1∣0⟩A2+ZS∣ψ⟩L∣Z⟩E∣0⟩A1∣0⟩A2\begin{split} \ket{\Psi}_3 \,=\quad &\ket{\psi}_L\ket{\mathbb{1}}_{E}\ket{0}_{A_1}\ket{0}_{A_2} + \ket{\psi}_L\ket{X}_{E}\ket{\color{violet}X_S}_{A_1}\ket{0}_{A_2}-\\ & Z_S\ket{\psi}_L\ket{Y}_{E}\ket{\color{violet}X_S}_{A_1}\ket{0}_{A_2} + Z_S\ket{\psi}_L\ket{Z}_{E}\ket{0}_{A_1}\ket{0}_{A_2} \end{split}∣Ψ⟩3​=​∣ψ⟩L​∣1⟩E​∣0⟩A1​​∣0⟩A2​​+∣ψ⟩L​∣X⟩E​∣XS​⟩A1​​∣0⟩A2​​−ZS​∣ψ⟩L​∣Y⟩E​∣XS​⟩A1​​∣0⟩A2​​+ZS​∣ψ⟩L​∣Z⟩E​∣0⟩A1​​∣0⟩A2​​​

At stage 4, each time there is a phase flip (Z), that information is stored in A2A_2A2​

∣Ψ⟩4 =∣ψ⟩L∣1⟩E∣0⟩A1∣0⟩A2+∣ψ⟩L∣X⟩E∣XS⟩A1∣0⟩A2−ZS∣ψ⟩L∣Y⟩E∣XS⟩A1∣ZS⟩A2+ZS∣ψ⟩L∣Z⟩E∣0⟩A1∣ZS⟩A2\begin{split} \ket{\Psi}_4 \,=\quad &\ket{\psi}_L\ket{\mathbb{1}}_{E}\ket{0}_{A_1}\ket{0}_{A_2} + \ket{\psi}_L\ket{X}_{E}\ket{X_S}_{A_1}\ket{0}_{A_2}-\\ & Z_S\ket{\psi}_L\ket{Y}_{E}\ket{X_S}_{A_1}\ket{\color{violet}Z_S}_{A_2} + Z_S\ket{\psi}_L\ket{Z}_{E}\ket{0}_{A_1}\ket{\color{violet}Z_S}_{A_2} \end{split}∣Ψ⟩4​=​∣ψ⟩L​∣1⟩E​∣0⟩A1​​∣0⟩A2​​+∣ψ⟩L​∣X⟩E​∣XS​⟩A1​​∣0⟩A2​​−ZS​∣ψ⟩L​∣Y⟩E​∣XS​⟩A1​​∣ZS​⟩A2​​+ZS​∣ψ⟩L​∣Z⟩E​∣0⟩A1​​∣ZS​⟩A2​​​

At stage 5, if A2A_2A2​ is in the ZSZ_SZS​ state, apply ZSZ_SZS​:

∣Ψ⟩5 =∣ψ⟩L∣1⟩E∣0⟩A1∣0⟩A2+∣ψ⟩L∣X⟩E∣XS⟩A1∣0⟩A2−∣ψ⟩L∣Y⟩E∣XS⟩A1∣ZS⟩A2+∣ψ⟩L∣Z⟩E∣0⟩A1∣ZS⟩A2\begin{split} \ket{\Psi}_5 \,=\quad &\ket{\psi}_L\ket{\mathbb{1}}_{E}\ket{0}_{A_1}\ket{0}_{A_2} + \ket{\psi}_L\ket{X}_{E}\ket{X_S}_{A_1}\ket{0}_{A_2}-\\ & \ket{\psi}_L\ket{Y}_{E}\ket{X_S}_{A_1}\ket{\color{violet}Z_S}_{A_2} + \ket{\psi}_L\ket{Z}_{E}\ket{0}_{A_1}\ket{\color{violet}Z_S}_{A_2} \end{split}∣Ψ⟩5​=​∣ψ⟩L​∣1⟩E​∣0⟩A1​​∣0⟩A2​​+∣ψ⟩L​∣X⟩E​∣XS​⟩A1​​∣0⟩A2​​−∣ψ⟩L​∣Y⟩E​∣XS​⟩A1​​∣ZS​⟩A2​​+∣ψ⟩L​∣Z⟩E​∣0⟩A1​​∣ZS​⟩A2​​​

The final state of the system L is ∣ψ⟩L\ket{\psi}_L∣ψ⟩L​.
Looking at the final state of EAEAEA, it is similar to the initial state LELELE, as if the correlations were transferred (like an entropy pump).
The 9 qubit code is not the most efficient, 5 qubits suffice for single qubit errors, see Minimal 1 qubit error code. Indeed, there are other errors aside from single qubit errors that are corrected by this code (e.g. two phase flip in the same triplets, 3 phase flips in the same triplets, 3 bit flip in the 3 different triplets).
Generally speaking, if we have nnn physical qubits, upon which a maximum number eee of simultaneous local errors can occur, there is a maximum number of logical qubits kkk which can be extracted, given by the condition
n≥4e+kn\geq 4e +kn≥4e+k
This picture can be reversed: if we need kkk logical qubits, and in our quantum computer a maximum eee of local errors happen at the same time, then the employed physical qubits are lower bounded by the relation above.
